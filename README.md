### Features
1. Pen & Eraser w/ adjustable width
2. Colour dropper
3. Shapes -> rectangle/curve/circle/line/oval/star
4. Magnifying glass
5. Fill tool w/ user-input precision levels
6. Generic N-sided polygon drawer
7. Select -> Move/Clear
8. Clear screen
9. Save/Load image onto canvas
